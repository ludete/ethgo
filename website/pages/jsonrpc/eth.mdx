
# Ethereum

## GetCode

Returns code at a given address.

```
code, err := client.Eth().GetCode(address, tag)
```

Params

- `address`: Address of the contract to query.
- `tag`: Either when to query the data.

Output

- `code`: Bytecode of the contract.
